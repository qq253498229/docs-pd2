新人MF法师

法师MF(Magic Find)要求很低，并且有TP(传送)可以极大提高效率。

适合前期快速MF日用品，一个法师养全家。

装备可以适当调整，前期有啥用啥；

不用太追求MF，一般堆到400左右就够了，再堆的话MF的收益就很低了。不如将FCR从63堆到105，或者多留点背包空间实在。

1. 装备

部位 名称 镶嵌 备注

武器

眼球

24#(MF)

没有就随便找个技能、FCR的杖

盾牌

警戒之墙

22#(全抗)

精神盾被削弱了

头盔

军帽

黄宝石(MF)

没有的话可以用3孔任意头盔，in3个黄宝石

项链

塔拉夏项链

不太好出，可以用马拉或者是其它MF、抗性、技能、FCR项链代替

衣服

塔拉夏衣服

黄宝石(MF)

不太好出，可以用IRE(诗寇蒂的愤怒)代替，实在没有也可以随便找个孔多的打满黄宝石(俗称玉米甲)

腰带

塔拉夏腰带

没有就用金色包袱，或者FHR+抗性黄腰带有FCR更好，或者+1技能的蛛网腰带

手套

运气守护

如果fcr不到63，优先用fcr手套凑到63档位

鞋子

战争旅者

或其他MF+抗性鞋，高跑+高抗鞋

戒指

两个拿各

如果fcr不到63，优先用fcr戒指凑到63档位；没有就用FCR+抗性+MF戒指

2. 背包

火炬、毁灭、抗性小符、7MF小符。

多留点空间，可以少去几次商店，提高效率。

3. 技能

针对不同场景选择不同技能。

MF安姐(和劳模一个掉落)就用火系，MF虫子(和巴尔一个掉落，强烈推荐)就用火/电。

前期好刷的85场景中，MF地穴用电，MF大陵寝用火球(跳到金怪脸上溅射)，MF古代通道用冰。

刚到地狱没装备可以用火墙卡位杀虫子，它打不到你，慢慢刮痧就行。

3.1. 火系

陨石自带OW(撕开敌人伤口)效果，可以防止BOSS回血，前期刷小黑很舒服。

  • 老三样: 温暖1-心灵传动1-传送1

  • 火系: 火支配max-陨石及协同max

陨石可以换成火球、火墙、九头海蛇。

3.2. 冰系

  • 老三样: 温暖1-心灵传动1-传送1

  • 冰系: 冰支配max-冰枪及协同max

冰枪可以换成暴风雪和冰封球。

3.3. 闪电

  • 老三样: 温暖1-心灵传动1-传送1

  • 闪系: 电支配max-闪电及协同max

4. 佣兵

前期米山扛不住就用京子或者可汗。

装备方面【卓古拉之握-长串之耳-蚀肉骑士】佣兵老三样；

武器用RW无限，开荒期间没符文就用RW眼光提供回蓝光环(弓也能做眼光)，或者丧钟、暴风尖塔(SF)、盗墓者、破骨；

头盔用吸血鬼的凝视(物免盔)，或者盗贼皇冠、安达利尔的面貌、年纪之冠；

衣服用谢夫特斯坦布(物免甲)，或者都瑞尔的壳、RW强制、RW石块、守护天使、圣堂武士的力量；

头盔衣服打孔插22#撑抗性。
