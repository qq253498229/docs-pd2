九头蛇大黑

  • 武器：脾气in火jew

  • 盾牌：暴风盾wmd，PDR20-30，瓦2s，in2个22#（22全抗）

  • 盔甲：扩散黑暗，撑抗性上限，瓦电抗上限，in完美红

  • 手套：法拳，瓦全抗

  • 头盔：2s年纪，PDR10-15，瓦火炕上限，in一个完美红（38life），一个8#（30fr）

  • 项链：塔拉夏

  • 靴子：水上漂，瓦电抗上限

  • 戒指：乌鸦（无法冰冻）+大自然（抗上限，pdr7-11）

  • 腰带：心结，10-15DR

技能：传送1，火焰专精MAX，巨九MAX，巨九加成MAX
