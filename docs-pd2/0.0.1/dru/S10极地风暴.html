S10基地风暴

https://www.youtube.com/watch?v=ps5mKEL6MbE

  • 武器：冰霜之风，in4冰猪，50冰冻

  • 盾牌：美杜莎盾，in3冰猪

  • 副手：CTA+警戒之墙

  • 头盔：夜翼面纱，in3冰猪，50冰冻

  • 盔甲：奥马斯，in3冰猪；

  • 盔甲：3技能衣服in冰珠

  • 手套：TGO，瓦30FCR

  • 腰带：冰雪腰带

  • 靴子：高跑3抗鞋

  • 项链：3元素蓝项链；

  • 项链：马拉

  • 戒指：鬼火，瓦10FCR;婚戒，瓦10FCR

技能：

  • 召唤系：右边两列一样1点，血灵max

  • 元素系：极地风暴max+龙卷风max+暴风max+飓风装甲max+风遁(剩下)

A2反抗光环

  • 武器：无限(7wc)

  • 头盔：RW残暴(33+32+13)，颚骨面甲底材(过滤器代码ba6,eth,3s)，提供嘲讽

  • 衣服：RW纯真(25+20+33+15)，高防底材(DEF800+)

  • 手套：卓古拉

  • 腰带：长串，提供PDR

  • 靴子：坦克雷套装绿鞋，提供PDR

或者用A1PET，用毒液怪打降低抵抗，其余装备全堆抗性保命
