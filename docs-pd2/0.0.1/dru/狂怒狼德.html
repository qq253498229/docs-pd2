  • 武器：地狱毁灭者-斩首斧eth，in13#*4，如果是eth需要in一个33#，瓦ll

  • 头盔：灵魂看守者-大地之灵，in26#，瓦生命上限

  • 盔甲：刚毅

  • 手套：碎钢，瓦降低目标防御

  • 腰带：雷神，瓦DR

  • 靴子：地狱阔步，瓦火炕上限

  • 项链：金属网格，瓦毒抗

  • 戒指：大自然+乌鸦，瓦准确

技能

  • 橡木智者max+狼獾之心max+食尸藤1

  • 狼人变化max+变形术max+狂怒max

  • 靴子：水上漂，瓦抗性上线

  • 项链：金属网格，瓦生命回复

  • 头盔：灵魂看守者-大地之灵，瓦3s，in2个26（火炕上限）,1个32（无法冰冻）

  • 盔甲：谢夫特斯坦布，瓦3s，in3个31（生命上线）

火炕90，电抗满，冰和毒抗尽量满

  • 靴子：吃肉鞋

42FHR，最好能到86FHR

参考视频: S9狂怒德鲁伊打L0大马

  • 武器：风锤-瓦ias/ED/6s-in13#(ias)无形需要一个33#(无法破坏)

  • 武器1：3s盗墓者in13#

  • 武器2：牛逼高伤武器-in13#-eth要in33#

  • 副手：CTA+警戒之墙

  • 头盔：灵魂看守者-瓦ed-in2个27#(最大冰抗)-10最大毒抗

  • 盔甲：荣耀之链(AP+底子)

  • 盔甲1：安全手工-in27#(最大冰抗)-ED-50%冰冻

  • 手套：碎钢-瓦ds

  • 手套1：死亡套装手套(毒缩75%)-瓦抗性

  • 腰带：蛇索(抗毒+毒缩50%)+大自然-3最大毒抗

  • 腰带1：FHR+抗性腰带-瓦pdr

  • 靴子：沙暴-瓦毒抗-3最大毒抗

  • 靴子1：吃肉-瓦抗性

  • 项链：打击手工项链毒缩75%+抗性、fhr、技能、准确越高越好
