新人刷T图强烈推荐：尸爆死灵

  • 尸暴的优势是既有火伤还有物伤，范围又特别大，NEC又自带物理盾能提升一定的生存。

  • 装备上可选项极多，既有暗金也有手工可以替换。

  • 这样就同时兼备了刷图效率、安逸和装备性价比，所以个人觉得是最适合新人刷T图的BD。

  • 属性方面优先125FCR保证施法速度，其次是4R提升安全保证刷图生存。再下来就是减抗和火猪继续提高输出、EK/MEK提高续航、FHR继续提高生存。最后还有DR提升

  • 物品栏火炬毁灭之外，既可以用毒骨大板子，也可以用火伤大符。

  • 个人推荐优先补满抗性，刷图更安逸。并少量堆一些EK/MEK，这样续航上去了节省买药次数，效率会提升很多。伤害方面随着提升会越来越高，不用太着急。

  • 下面的装备仅供参考，根据自己的情况进行替换。

1. S11尸暴

  • 武器：破隐,in2个火猪

  • 盾牌：打击手工盾，2死灵+3尸爆

  • 头盔：齐拉守护，15+3*5=30减抗

  • 盔甲：魁黑刚，3*5=15减抗

  • 腰带：技能腰带

  • 手套：霜燃

2. 属性

大板子用毒骨，中板子用火伤，小符补抗+血

  • 减抗：武器20+6*5+头盔15+3*5+项链8+衣服3*5+手套10=113，加上无限34一共147

  • 尸爆伤害9000+

  • FCR堆到125

  • FHR越高越好，最好堆到86

3. S9尸爆

  • 武器：施法手工杖，2死灵+3尸爆+1技能+瓦1技能+fcr

  • 盾牌：打击手工盾，2死灵+3尸爆+瓦无法冰冻

  • 头盔：2死灵+20FCR黄头盔+in2个火猪，有抗性、ek最好

  • 盔甲：打击手工甲，1死灵+FHR+瓦1技能+in13#和火猪

  • 腰带：施法手工腰，20FCR+瓦10FCR

  • 手套：施法手工手，20FCR+瓦10FCR+EK

  • 靴子：3抗黄鞋+ek

  • 项链：3毒骨蓝项链+瓦1技能

  • 戒指：10FCR黄戒指+瓦10FCR，2个，有抗性、ek最好

4. S8尸爆

  • 武器：梅格之歌的教训，85fcr，20+6*5=50减抗

  • 头盔：齐拉守护，15+3*5=30减抗

  • 盔甲：魁黑刚，3*5=15减抗

  • 手套：法拳

  • 腰带：心结

  • 靴子：3抗鞋

  • 项链：旭日东升，8减抗

  • 戒指：soj*2，或者用鬼火技能戒指招狼獾灵

5. S8尸爆

  • 武器：破隐+瓦FCR+in火猪

  • 盾牌：黄尿壶+2nec+3尸爆+瓦技能+in火猪

  • 头盔：黄头环+2nec+20fcr+瓦3孔+in火猪

  • 盔甲：亚特玛的哭喊+瓦3孔+in火猪

  • 腰带：技能腰带+瓦fcr

  • 手套：法拳+瓦fcr

  • 靴子：3抗黄鞋

  • 项链：2毒骨+fcr+瓦无法冰冻

  • 戒指：soj+鬼魂

6. 技能

  • 优先点1点TP和1点骨甲，然后点满尸爆和联动技能，剩下的加到TP上，最后还有多的加到骨甲上

  • 召唤：复生max+血次元max

  • 诅咒：伤害加深max+诅咒精通max

  • 毒骨：骨甲1+尸爆max+亵渎max

7. 佣兵

  • 佣兵武器：无限

  • 佣兵头盔：摇曳焰火

  • 佣兵衣服：圣堂

  • 佣兵手套：qh手套

  • 佣兵腰带：长串

  • 佣兵靴子：吃肉
