= 暗黑中的黑话整理(术语、缩写)

暗黑玩家之间的交流经常出现一些英文单词缩写、术语、黑话。

其原因最开始是因为游戏不支持中文输入法，迫使玩家用必须英语交流； 后来虽然有一些插件能让玩家输入中文，但是玩家已经习惯这些交流术语了，并且这些术语通常更容易输入，所以就被延续下来了。

新人刚刚接触暗黑时，经常会被这些乱七八糟的英文给搞的晕头转向，例如：

* buy 940def 7+
* buy glf
* sell 10fcr 4r15 ring

所以在这里整理一些常见的术语、缩写，提供给玩家查询。欢迎补充。

== 数字

* 1（bus用语，安全）
* 2（bus用语，危险）
* 3bar/3BB（指act5 q5的三个野蛮人守护神）
* 3q（Thank you)
* 40/15（指40ed/15ias jewel或者40ed/15max damage jewel）
* 5小队（杀baal途中，baal的5次召唤怪物）
* 7＋（漆甲，通常指塔拉夏的守护套装盔甲）
* (X)S（X个孔，例如: 4s armor就是4孔盔甲）
* (X)\#（符文编号，28# 就是28号符文Lo）
