= 常用Buff整理

== 祈祷

提供生命恢复和击中回复生命

* A3冰冻佣兵附带光环，最高18级

== 抵抗火焰

6级提供10%火焰伤害45%抗火

* 暗金精英单手剑，魔灵杀手，提供6级抵抗火焰灵气
* 暗金拓展双手剑，特迪斯法雷-芙法米，提供6级抵抗火焰灵气
* RW3孔头盔(4+21+26)，摇曳烟火，提供1-6级抵抗火焰灵气

== 抵抗冰冷

6级提供10%冰冷伤害45%抗冰

* 暗金精英单手剑，冰霜之风，提供6级抵抗冰冷灵气

== 抵抗闪电

6级提供10%闪电伤害45%抗电

* 暗金精英单手剑，光之军刀，提供6级抵抗闪电灵气

== 反抗

13级提供128%防御以及6%减少物理伤害；16级提供146%防御以及8%减少物理伤害；18级提供162%防御以及9%减少物理伤害

* A2防御佣兵附带光环，最高18级
* RW流亡13-16级

== 净化

* A3佣兵
* RW瘟疫15-17级

== 活力

* RW和谐10级

== 冥思

== 救赎
