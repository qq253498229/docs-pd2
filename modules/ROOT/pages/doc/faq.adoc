= 新人常见问题

== 镶嵌快捷键是什么？我怎么镶嵌不了？如何镶嵌宝石、符文？

符文和其他可以镶嵌类的物品需要转成“非堆叠”形态才能镶嵌，方法：

* 1、先把“堆叠形态”的“符文拿起来”，按住“ctrl+shift”再按“鼠标左键”可以放下1个“非堆叠”形态的符文。
* 2、先把“堆叠形态”的“符文拿起来”，按住“ctrl”再按“鼠标左键”可以放下1个“堆叠"形态的符文。
* 3、“1个堆叠或者非堆叠”的符文可以通过“ctrl+shift”再按“鼠标左键”来互相转换。（不用拿起来）
* 4、游戏内按“H”键可以查看上述方法说明以及其他常用说明。
