= 开荒简略流程

* 开荒剧情，点满主要输出技能
* 噩梦MF为进军地狱做准备
* 地狱MF，升级到80级满足开T图要求
* 高效MF虫子或直接开刷T图
** 虫子使用的是巴尔的掉落，安姐使用的是劳模的掉落
* MF出装腐化，做RW及手工，目标核心BD。
* 刷小黑：刷虫子出的暗金/绿装卖商店，有5%几率出小黑，出高级符文(最大28#)和手工材料
* 刷T3+亮金+腐化地图
* 刷3BB（大量暗金）
* 刷6BOSS（火炬）
* 刷BOSS：大黑（DC）毁灭小符
* 刷最终BOSS：拉斯玛（DC腐化石）
