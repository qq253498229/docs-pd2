= 经验获取

在暗黑2中怪物经验的获取是根据人物于怪物等级的差值计算的，差值过大则会有经验惩罚。 简单的说就是尽量在适当的等级打适当的怪。

.在25级之前
[%autowidth]
|===
|差距等级 |获得经验%
|0-5 |100
|6 |88
|7 |68
|8 |36
|9 |15
|10或10以上 |2
|===

.在25级之后
[%autowidth]
|===
|差距等级 |获得经验%
|0-5 |100
|6 |81
|7 |62
|8 |43
|9 |24
|10或10以上 |5
|===

NOTE: 表格不一定准确，大致参考一下
