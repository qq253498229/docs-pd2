= 死灵T图:S10淬毒攻击

原视频地址: https://www.youtube.com/watch?v=pgE5MIZAeSY

== 装备

* 武器：RW瘟疫(32+19+22)，技能底材(过滤器代码SK73>0,3s,最好+3淬毒攻击SK73)，提供5技能+20FCR+减毒抗+降低抵抗+净化光环
* 盾牌：骨焰(nee)，提供3nec技能+最大20PDR+30R+8LEK+30FRW，in1个毒珠+2个ias抗性珠
* 头盔：安头，瓦3s，in1个毒珠+2个ias抗性珠
* 盔甲：RW野蔷薇(8+27+29+5)，AP+底材(过滤器代码utp,4s)，提供50毒伤+50FHR+6LEK
* 手套：TGO套装手，瓦30FCR
* 腰带：技能腰带，瓦30FCR
* 靴子：水上漂
* 项链：大君，提供20ias
* 戒指：乌鸦;鬼火，瓦10FCR

== 技能

* 召唤：血次元到30+支配石魔max+复生(剩余)
* 毒骨：淬毒攻击max+亵渎max+骨甲max

== 综合

* 力量敏捷够穿装备，剩下加法，这样骨甲护盾更高
* 75FCR:武器20+腰带30+手套30+戒指10
* 86FHR:衣服50+3个12FHR大板
* 减毒抗30:武器20+头盔珠子5+盾牌珠子5
* ias99(10f):盾牌珠子30+项链20+头盔30+头盔珠子30
* 背包除了火炬毁灭外，带满毒骨大板，最好带3个12FHR的凑到86FHR，其它大板用+45血的，最后剩余位置用小符补抗
* 进战斗招5个血魔，复生怪招满，`*保持骨甲*`，血次元跳到怪中间用淬毒攻击，注意毒伤有延迟，不用等怪死了就可以飞走

== 佣兵

PET很灵活，可以用A5呐喊PET吼血

* 武器：RW狂喜，食人魔之锤底材(过滤器代码7m7,eth,6s)，提供降低抵抗
* 头盔：RW残暴，颚骨面甲底材(过滤器代码ba6,eth,3s)，提供嘲讽
* 衣服：RW纯真(25+20+33+15)，高防底材(DEF800+)
* 手套：熔岩角羊
* 腰带：长串，提供PDR
* 靴子：坦克雷套装绿鞋，提供PDR

或者用A1PET，用毒液怪打降低抵抗，其余装备全堆抗性保命

如果攻速不够带狂热光环补，例如A4黑暗PET配野兽杖、A1活力PET配信心弓等
