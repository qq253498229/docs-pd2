= 死灵T图:S10火魔

原视频地址: https://www.youtube.com/watch?v=zmP7VrF_wvs

== 装备

* 武器：RW圣域(32+27)，技能底材(过滤器代码SK94>0,2s,最好+3火焰石魔SK94)
* 盾牌：RW凤凰(26+26+28+31)，统盾底材(过滤器代码uit,4s)，提供28减火炕+lm+救赎光环
* 头盔：军帽，in2个22#，瓦技能
* 盔甲：RW谜团(31+6+30)，MP+底材(过滤器代码xtp,3s)，提供聚气传送
* 手套：法师之拳，瓦30FCR
* 腰带：技能腰带
* 靴子：BUG鞋，提供无法冰冻
* 项链：3召唤蓝项链
* 戒指：鬼火，瓦10FCR;婚戒，瓦10FCR

== 技能

* 召唤：支配石魔max+4石魔max+血次元进战斗30+复生1
* 毒骨：骨甲1

== 综合

* 力量敏捷够穿装备，剩下加法，这样骨甲护盾更高
* 75FCR:武器25+手套30+腰带20+戒指10+戒指10
* 背包除了火炬毁灭外，带满召唤大板最好+45血的，最后剩余位置用小符补抗
* RW无限铁魔
* 进战斗招5个火魔，`*保持骨甲*`，TP/血次元到怪外围等火魔打
