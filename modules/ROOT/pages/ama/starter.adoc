= 亚马逊开荒流程

== 流程

* 开局加戳刺；
* 6级洗点改用威力一击；接下来优先加威力一击，多的点数加闪电球，一直能用到35级。
* 快速过剧情，把必须的任务都做了，15级左右到A2，级别太低的话去崔斯特瑞姆刷点怪；
* 23级左右到A3，级别太低就去塔拉夏古墓刷点怪。
** A1火焰雇佣兵在18级开启活力光环，极大增加移动速度，在20级左右比较容易刷到18级的雇佣兵，第一时间换上
* 快速过剧情，一直到A5第一个传送点，刷矫正者和督军山克升级到35级以上。期间慢慢增加pp数，保证效率的情况下到8pp没问题。
* 快速通过普通难度。
** 在35级左右的时候，洗点换成召唤流，记得刷矫正者和督军山克仍然是威力一击更效率，级别够了再洗。
** 用女武神作为主力输出，见怪先放诱饵，做出和平甲后能招3个女武神。
* 29级可以穿和平甲(13+10+11)可以多招一个女武神，39级可以穿和谐弓(3+6+12+18)活力光环，替换掉活力京子改用其他雇佣兵的光环。推荐A2反抗米山增加生存能力。
** 和平甲优先做出来，和谐弓不着急，看运气能出最好，不行的话到地狱再刷刷女伯爵就行。
* 到噩梦之后迅速过剧情到A5，还是第一个传送点处升级到58级左右，这时候过完剧情就差不多60级了，然后去地狱。

== 装备符文之语

* 13级能用天底头(4+3)增强回蓝，可以少喝几个蓝屏，非必要；
* 17级能用隐秘甲(7+5)增加RUN/FCR/FHR以及毒抗，非常推荐；
* 21级能用先祖盾(8+9+7)增加巨量抗性，A5任务正好给一套符文，随便捡个3孔盾打上，推荐；
* 25级能用精神(7+10+9+11)剑，在洗成召唤流之后，没有好用武器的话可以做一个；
* 27级能用知识头(9+12)，+1技能；
* 29级能用和平甲(13+10+11)，+2亚马逊技能，额外找一个女武神
* 30级能用和谐弓(3+6+12+18)，活力光环，大量女武神技能
* 也可以做手工，比如施法手工武器(3#+紫)固定1技能+FCR+mana。


== 技能

* 被动：女武神max-诱饵max-刺入max
* 弓弩：炮轰max

== 总结

利用阿卡拉无限洗点的特性，挑选不依赖装备的强力技能，可以极大提高开荒效率。

注意经验获取和人物等级、地图等级的关系，避免浪费经验的情况。

大部分流程其他职业也能用，除了职业专有的部分，比如和平甲和和谐弓。
