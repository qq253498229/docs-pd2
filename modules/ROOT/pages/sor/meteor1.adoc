= 新人MF法师

法师MF(Magic Find)要求很低，并且有TP(传送)可以极大提高效率。

适合前期快速MF日用品，一个法师养全家。

装备可以适当调整，前期有啥用啥；

不用太追求MF，一般堆到400左右就够了，再堆的话MF的收益就很低了。不如将FCR从63堆到105，或者多留点背包空间实在。

== 背包

火炬、毁灭、抗性小符、7MF小符。

多留点空间，可以少去几次商店，提高效率。

== 技能

针对不同场景选择不同技能。

MF安姐(和劳模一个掉落)就用火系，MF虫子(和巴尔一个掉落，__**强烈推荐**__)就用火电。

前期好刷的85场景中，MF地穴用电，MF大陵寝用火球，MF古代通道用冰。

刚到地狱没装备可以用火墙卡位杀虫子，它打不到你，慢慢刮痧就行。

=== 火系

陨石自带OW(撕开敌人伤口)效果，可以防止BOSS回血，前期刷小黑很舒服。

- 老三样: 温暖1-心灵传动1-传送1
- 火系: 火支配max-陨石及协同max

陨石可以换成火球、火墙、九头海蛇。

=== 冰系

- 老三样: 温暖1-心灵传动1-传送1
- 冰系: 冰支配max-冰枪及协同max

冰枪可以换成暴风雪和冰封球。

=== 闪电

- 老三样: 温暖1-心灵传动1-传送1
- 闪系: 电支配max-闪电及协同max

== 佣兵

前期米山扛不住就用京子或者可汗。

装备方面【卓古拉之握-长串之耳-蚀肉骑士】佣兵老三样；

武器用RW无限，开荒期间没符文就用RW眼光提供回蓝光环(弓也能做眼光)，或者丧钟、暴风尖塔(SF)、盗墓者、破骨；

头盔用吸血鬼的凝视(物免盔)，或者盗贼皇冠、安达利尔的面貌、年纪之冠；

衣服用谢夫特斯坦布(物免甲)，或者都瑞尔的壳、RW强制、RW石块、守护天使、圣堂武士的力量；

头盔衣服打孔插22#撑抗性。
